Automation

Collection Queue

Paste one Sugargoo seller/product URL, 1688 offer URL, Taobao item URL, or CSSBuy item URL per line. The queue opens a local browser, resolves product listings to their seller page, collects the shop, saves a fresh manifest.json plus images into scraped data/<shop>, and refreshes this browser when each run completes. You can keep adding more URLs while the queue is already running. If Sugargoo asks for sign-in, use the opened collection browser window once and the queue will continue automatically.
